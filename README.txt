Name of application: CRUD 2.0
Author: Raymacz
Version: 1

How to use: 

---ADD---
1. Input the following
-Name (required)
-Phone Number (required) 
-Address (optional)
-Email (optional)
-Civil Status (Single option is initially selected)
2. Click on ADD button

---EDIT---
1. On the contacts table click on Edit link.
2. Update desired info.
2. Click on UPDATE button.

---DELETE---
1. On the contacts table click on Delete link.
2. A confirmation box pops up and click on OK to confirm deletion.
